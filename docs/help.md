```bash
/help ............... Display help information — 顯示說明資訊
/init ............... Configure project for better results — 為獲得更佳結果設定專案
/login .............. Authenticate your session — 驗證您的會話/身份
/logout ............. Sign out of your session — 登出您的會話
/diff ............... Show last assistant change diff — 顯示助理上次變更的差異
/checkpoint <id> .... Restore to a specific checkpoint — 還原至指定的檢查點（<id>）
/usage .............. View remaining / bonus AI credits — 檢視剩餘／獎勵 AI 點數
/reset .............. Reset the conversation history, as if you just started a new Codebuff session — 重設對話紀錄，彷彿開始新的 Codebuff 會話
/compact ............ Replace the conversation history with a summary and free up context — 以摘要取代對話紀錄並釋放上下文
/export ............. Export conversation summary to file — 將對話摘要匯出為檔案
/undo ............... Undo last change — 還原上一次變更
/redo ............... Redo last undone change — 取消還原（重做）上一次變更
/ask ................ Switch to ask mode (won't modify code) — 切換到詢問模式（不會修改程式碼）
/lite ............... Switch to lite mode (faster, cheaper) — 切換到精簡模式（較快、成本較低）
/normal ............. Switch to normal mode (balanced) — 切換到正常模式（平衡）
/max ................ Switch to max mode (slower, more thorough) — 切換到最大模式（較慢、較徹底）
/trace [trace-id] ... Show trace list or view specific trace by ID — 顯示追蹤清單或以 ID 檢視特定追蹤
/agents ............. Manage custom agent templates in .agents/ — 管理 .agents/ 中的自訂代理範本
/exit ............... Quit Codebuff — 離開 Codebuff
Tip: Type "!" followed by a command to run it in your shell, e.g., !ls — 提示：在 shell 中於命令前加上「!」來執行，例如：!ls
```
