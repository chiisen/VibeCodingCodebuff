# VibeCodingCodebuff
Vibe Coding Codebuff - AI 驅動的程式碼助手，使用自然語言編輯和改進您的程式碼庫。

![codebuff-vs-claude-code](./images/codebuff-vs-claude-code.png)

# 安裝
Node.js 20
```bash
npm install -g codebuff
```

# 啟動
```bash
cd your-project
codebuff
```

```bash
依據 https://github.com/CodebuffAI/codebuff 的內容的彙整製作一頁式的介紹說明網頁，網頁風格請參考 STYLE.md。
```

網頁特色包括：
   • 🎨 現代漸層設計（紫色系）
   • 📱 響應式佈局，支援行動裝置
   • 🚀 包含所有核心功能介紹
   • 💻 安裝與使用範例
   • 📊 效能數據展示
   • ✨ 流暢的懸停動畫效果
您可以直接在瀏覽器中開啟 `index.html` 檔案來查看網頁效果！
